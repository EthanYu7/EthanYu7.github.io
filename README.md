# 2026马年新春烟花互动页面

一款基于 HTML5 Canvas + JavaScript 开发的马年新年主题互动页面，主打倒计时、互动烟花与沉浸式视觉体验，适配多端设备。

## ✨ 核心功能

1. **新年倒计时**：以2026年2月17日（农历新年）为目标，实时显示剩余时间，新年后自动切换祝福文案。

2. **开场动画**：新年到达后，自动播放数字倒计时+新春祝福逐句展示动画，搭配鎏金光影效果。

3. **互动烟花**

    - 点击/触摸屏幕：燃放1个烟花（30%概率为马年祝福语烟花）

    - 长按屏幕：连续发射普通粒子烟花

    - 自动燃放：每2.5秒随机位置自动发射1枚烟花

    - 两种类型：普通粒子烟花（含马年元素）、祝福语烟花（8种马年祝福）

4. **音效控制**：右上角静音按钮，可切换烟花爆炸音效的播放/静音状态。

5. **多端适配**：支持手机、平板、电脑，全屏沉浸式展示，适配不同分辨率。

## 💻 技术栈

HTML5 + CSS3 + 原生JavaScript + Canvas 2D + Web Audio API

## 📖 使用说明

1. 打开页面即可自动加载，进入倒计时或开场动画。

2. 点击/长按屏幕任意位置，触发烟花燃放。

3. 点击右上角喇叭图标，控制音效开关。

4. 建议全屏模式体验，禁止页面缩放，提升视觉效果。

## ⚙️ 性能限制

- 普通粒子烟花最大同时存在：10个

- 祝福语烟花最大同时存在：3个

- 总烟花上限：15个（避免性能占用过高）
